# <center>__Pseutocode__</center>

* ## What is Pseutocode?
* ### *__Pseutocode__* is CLI software used for converting *__Pseudocode__* from .txt files into any Programming language (that is supported) by *__Artificial Intelligence__* and writes in the language's correct file (like .py for Python).

<center>====================</center>

* ## How does Pseutocode work?
* ### Generally, The software checks for Pseudocode file (.txt) and handles file errors, If there are no errors, Then go next.
* ### Connects to Groq API (with my/your GROQ_API_KEY) and the AI analyzes the logic of the Pseudocode, Then.
* ### Writes in in the suitable language's file (due to dictionary, SQLite database in next version).

<center>====================</center>

* ## Pseutocode can be used in:
* ### Education (High school or college)
* ### Converting your Math/Algorithms into code
* ### Make your softwares more accurate by your own logic instead of prompt
